# ArcChat.UI.Controls

Reusable Avalonia control primitives for NC03.

## Style Selectors

| Control | Selector | Purpose |
| --- | --- | --- |
| `IconButton` | `controls\|IconButton` | Icon-first command button. |
| `IconButton` | `controls\|IconButton[IsPrimary=True]` | Primary action styling. |
| `IconButton` | `controls\|IconButton[IsDanger=True]` | Destructive action styling. |
| `IconButton` | `controls\|IconButton[IsBordered=True]` | Bordered icon button. |
| `TextField` | `controls\|TextField` | Settings/search text input wrapper. |
| `Sidebar` | `controls\|Sidebar` | Shell sidebar host and nav list. |
| `Sidebar` | `controls\|Sidebar[IsNarrow=True]` | Icon-oriented narrow sidebar state. |
| `ResizableSplitPane` | `controls\|ResizableSplitPane` | Two-pane splitter primitive. |
| `ConfirmDialog` | `controls\|ConfirmDialog` | Reusable confirmation dialog. |
| `LoadingSpinner` | `controls\|LoadingSpinner` | Deterministic loading placeholder. |
| `EmptyState` | `controls\|EmptyState` | Empty-list/detail placeholder. |
